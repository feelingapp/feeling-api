## How To Use

Install Python libraries (for formatting):

```bash
python3 -m venv .env
source .env/bin/activate
pip install -r requirements.txt
```

Install JavaScript libraries (for Serverless plugins):

```bash
npm install
```

To run the functions locally:

```bash
npm run develop
```
